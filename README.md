# Heritage Motor Museum

A web-based application built using **Spring Boot** that showcases a Heritage Motor Museum.  
The project provides user authentication and serves static web pages along with REST APIs.

---

## Technologies Used
- Java 17  
- Spring Boot 3.x  
- Spring Security  
- Maven  
- MySQL  
- HTML, CSS, JavaScript  

---

## Features
- User Registration and Login
- Secure authentication using Spring Security
- RESTful APIs
- Static web pages served from the backend
- MySQL database integration
